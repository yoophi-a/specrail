# MVP Roadmap

## Milestone 0 — Scaffold
- workspace bootstrap
- domain package skeleton
- adapter package skeleton
- API app placeholder
- architecture docs

## Milestone 1 — Artifact-first control plane
- create project records
- create track folders and metadata
- write `spec.md` / `plan.md`
- update track indexes

## Milestone 2 — Single execution bootstrap
- create execution metadata
- allocate local workspace path
- add one adapter stub with spawn/resume lifecycle
- append JSONL events

## Milestone 3 — Approval checkpoints
- service-managed approval requests
- plan/spec approval persistence
- approval events in execution log

## Milestone 4 — Resume and summaries
- follow-up execution endpoint
- resumable adapter contract
- final summary generation contract

## Milestone 5 — Second adapter + GitHub hooks
- add another provider adapter
- emit commit/branch summaries
- optional GitHub integration spike
