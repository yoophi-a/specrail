# SpecRail Project Index

This directory is the control-plane artifact root for a managed project.

## Files
- `product.md` — product summary and goals
- `product-guidelines.md` — product constraints and review rules
- `tech-stack.md` — technical stack and conventions
- `workflow.md` — execution and approval workflow
- `tracks.md` — track registry
- `tracks/<track_id>/` — per-track specs, plans, metadata, and events
